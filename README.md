# AlimenTar — Sistema de Gestão de Doações

Projeto acadêmico desenvolvido na disciplina Projeto Integrador Extensionista I (PIE I) — 2026.

## Sobre o projeto

Sistema web para ajudar a ONG AlimenTar, localizada em Paraguaçu Paulista – SP, a organizar suas doações e beneficiados. Hoje tudo é feito no papel e em planilhas, o sistema vai centralizar essas informações em um só lugar.

## Tecnologias utilizadas

- Node.js + Express (backend)
- React ou HTML/CSS/JS (frontend)
- MySQL (banco de dados)
- Sequelize (ORM)

## Como rodar

Em construção — será preenchido no PIE II quando o código for desenvolvido.
